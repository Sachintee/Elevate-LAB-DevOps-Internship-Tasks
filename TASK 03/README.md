## Objective
Provision a Docker container using Terraform and understand IaC concepts.

## Steps
1. Installed Terraform and Docker.
2. Created Terraform configuration (main.tf).
3. Ran `terraform init` to install provider.
4. Ran `terraform plan` to preview infra.
5. Ran `terraform apply` to create Docker container.
6. Verified using `docker ps`.
7. Accessed container at http://localhost:8080.
8. Destroyed infra using `terraform destroy`.
